A modern, responsive dashboard for high-level PvP combat analysis and weapon reference.

## Core Features
- **Combat Calculator**: Real-time conflict resolution logic based on Stagger and Defense ratings.
- **Weapon Reference**: Centralized database of meta-standard priorities for all weapons and mystic skills.
- **Synergy Documentation**: Detailed "Important Information" tab documenting complex mechanics (Attunements, Inner Ways, and conditional buffs).

## Tech Stack
- **Frontend**: Vanilla HTML5, CSS3, and JavaScript.
- **Design**: Premium "Glassmorphism" aesthetic with a responsive, mobile-first grid layout.
- **Data-Driven**: Dymanically populated UI components sourced from a centralized JS data object.

## Metadata Guidelines
Statistics throughout the site assume a "Competitive Standard" (e.g., specific Inner Way buffs or common Attunements) to ensure calculation accuracy for advanced players.
